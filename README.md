# Keto Lite PWA

Чистое кето-приложение для iPhone: дневник еды + база продуктов + облачное сохранение через GitHub Gist.

## Что есть

- Две вкладки: Дневник и База.
- Калории, белки, жиры, углеводы.
- Ручная база продуктов на 100 г.
- Добавление еды по граммам.
- Дневной статус кето по углеводам.
- Вставка продукта от ChatGPT через JSON.
- Локальное хранение на телефоне через localStorage.
- Облачный бэкап в GitHub Gist.
- PWA для GitHub Pages и установки на iPhone.

## Облако GitHub Gist

1. Создайте GitHub token с доступом к Gist.
2. Откройте приложение → База → Облачное сохранение.
3. Вставьте token.
4. Нажмите «Сохранить настройки облака».
5. Нажмите «Выгрузить». Приложение само создаст секретный Gist и сохранит его ID.
6. На другом устройстве вставьте тот же token и Gist ID, затем нажмите «Загрузить».

Токен хранится только в localStorage текущего устройства. Не публикуйте token в GitHub-репозитории и не отправляйте его никому.

## Обновление на GitHub Pages

Загрузите файлы из папки в корень репозитория: index.html, app.webmanifest, sw.js, icons/.

После обновления на iPhone откройте сайт в Safari, обновите страницу 1-2 раза и при необходимости заново добавьте на экран Домой.
