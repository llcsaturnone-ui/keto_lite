# Простой трекер питания v9

Что добавлено:
- Новая вкладка `Ритм` для ежедневных привычек, намазов, тренировок и шагов.
- Шаги хранятся по дням, цель шагов можно менять.
- Привычки можно отмечать, добавлять и удалять.
- У каждого намаза теперь две отметки: намаз и зикр (суохьаж).
- Тренировки стали проще: отжимания, приседания, подтягивания, планка и пресс.
- Backup теперь включает питание, продукты, цели и данные вкладки `Ритм`.

Что улучшено:
- Отрицательные граммы, КБЖУ и значения продуктов больше не попадают в расчёты.
- Backup обновлён до версии 9, старые backup v7/v8 остаются совместимыми.
- В шапке приложения появилась отдельная метрика для вкладки `Ритм`.
- Кнопки с иконками получили доступные подписи для браузера и скринридеров.
- Вкладки `Дневник` и `База` визуально приведены к стилю вкладки `Ритм`.
- Нижняя навигация закреплена к низу окна, а iPhone Safari больше не должен оставлять пустоту снизу после открытия и закрытия клавиатуры.
- Поля ввода на iPhone поднимаются над клавиатурой, чтобы было видно, что редактируется.

Что стоит улучшить дальше:
- Для продакшена лучше собрать React/Tailwind локально, а не грузить их через CDN и Babel прямо в браузере.

Формат для массового добавления продуктов:
```json
[
  {"name":"Мандарин","calories":53,"protein":0.8,"fat":0.3,"carbs":13.3},
  {"name":"Киви","calories":61,"protein":1.1,"fat":0.5,"carbs":14.7}
]
```

Открой `index.html` в браузере.


## Иконка приложения

Добавлены файлы:
- `icon-180.png` — иконка для iPhone / iOS.
- `icon-192.png` — иконка для Android/PWA.
- `icon-512.png` — крупная иконка для Android/PWA.
- `manifest.webmanifest` — manifest для PWA.

В `index.html` добавлены только строки в `<head>` для подключения иконок. Логика приложения не менялась.

Если хочешь свою иконку, просто замени эти PNG-файлы на свои с такими же именами.
На iPhone после замены иконки нужно удалить старую иконку с экрана «Домой» и заново добавить сайт через Safari → Поделиться → На экран «Домой».
