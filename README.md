# Простой трекер питания v3

## Новое

- Добавлен надёжный импорт продукта через JSON.
- Можно вставить JSON прямо из ответа ChatGPT:
  {"name":"Яйцо варёное","calories":155,"protein":12.6,"fat":10.6,"carbs":1.1}
- Старый режим распознавания обычного текста тоже оставлен.

## Как использовать

Во вкладке «База» вставь JSON в блок «Вставить продукт JSON / из ответа ChatGPT» и нажми:
- «Добавить сразу»;
- или «Заполнить поля», проверить и сохранить.

## Установка

Залей `index.html` в GitHub Pages вместо старого файла.
