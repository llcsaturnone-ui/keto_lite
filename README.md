# Простой трекер питания v6

Новое:
- Одним JSON можно добавить несколько продуктов.
- Поддерживается массив:
[
  {"name":"Мандарин","calories":53,"protein":0.8,"fat":0.3,"carbs":13.3},
  {"name":"Киви","calories":61,"protein":1.1,"fat":0.5,"carbs":14.7}
]

Также поддерживается:
- один объект JSON;
- объект с products/items/foods;
- старый обычный текст.

Кнопка «Добавить сразу» добавляет все продукты.
Кнопка «Заполнить поля» берёт первый продукт.
