# Простой трекер питания v2

Версия в стиле старого приложения.

## Что есть

- Только диета, без тренировок, шагов, веса и талии.
- Дневник еды по датам.
- Калории, белки, жиры, углеводы.
- Прогресс по КБЖУ.
- Продукты с КБЖУ на 100 г.
- Добавление, редактирование и удаление продуктов.
- Редактирование съеденной еды за день.
- Backup JSON для анализа в ChatGPT.

## Новое в v2

- Добавлена функция «Вставить продукт из ответа ChatGPT».
- Можно вставить мой ответ с КБЖУ, нажать «Добавить сразу» — продукт появится в базе.
- Можно нажать «Заполнить поля», проверить данные и затем добавить вручную.

## Как установить

Залей `index.html` в GitHub Pages вместо старого файла.
