# КБЖУ + Тренировки

Простое веб-приложение без React, Tailwind и внешних библиотек.

## Что умеет

- дневная норма: ккал, белки, жиры, углеводы, шаги;
- добавление еды по граммам;
- автоматический подсчёт остатка нормы;
- база продуктов с КБЖУ на 100 г;
- тренировки на главном экране;
- шаги, вес, талия, заметка дня;
- сохранение в localStorage;
- экспорт/импорт backup JSON;
- PWA: можно добавить на экран iPhone.

## Как залить на GitHub Pages

1. Создай новый репозиторий на GitHub.
2. Загрузи все файлы из этой папки в корень репозитория:
   - index.html
   - styles.css
   - app.js
   - manifest.json
   - sw.js
   - папку icons
3. Открой Settings → Pages.
4. Source: Deploy from a branch.
5. Branch: main, folder: /root.
6. Сохрани.
7. Через минуту GitHub даст ссылку на сайт.

## Как установить на iPhone

1. Открой сайт через Safari.
2. Нажми «Поделиться».
3. Выбери «На экран Домой».
