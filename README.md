# КБЖУ Трекер v12

Готовый однофайловый вариант в стиле старого приложения.

## Главное

- Вкладки: Главная / Еда / Тренировка.
- Главная только показывает данные.
- Еда: добавление еды, база продуктов, редактирование/удаление продуктов, данные дня.
- Тренировка: отдельный прогресс по каждому упражнению, редактор шаблона, данные дня.
- Цели КБЖУ копируются с ближайшей предыдущей даты, пока ты их не изменишь.
- Тренировочный шаблон копируется на новую дату.
- Backup JSON можно скачать и отправить в ChatGPT для анализа прогресса.

## Как заменить

Залей `index.html` вместо старого файла в GitHub Pages.
