# КБЖУ Трекер — old style

Однофайловое React-приложение в стиле старого трекера.

## Что добавлено

- КБЖУ полностью: калории, белки, жиры, углеводы.
- Остаток дневной нормы.
- Поиск продукта при добавлении еды.
- База продуктов с КБЖУ на 100 г.
- Шаги, вес, талия и заметка дня.
- Отдельная вкладка тренировок.
- Дневной план: минимум дня, ходьба, силовая база.
- Backup JSON: скачать и загрузить.
- Адаптация под iPhone/PWA.

## Как использовать на GitHub Pages

1. Залить `index.html` в корень репозитория.
2. Settings → Pages → Deploy from branch → main / root.
3. Открыть ссылку в Safari.
4. Поделиться → На экран «Домой».
