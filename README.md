# Простой трекер питания v5

Проверка и исправления:
- Добавлена защита от повреждённых данных в localStorage: приложение не должно падать при битом JSON.
- Исправлена смена даты через локальную дату, без возможного сдвига из-за timezone.
- Добавлена поддержка JSON с вложенными полями `per100g`, `nutrition`, `macros`.
- При добавлении продукта с уже существующим названием теперь предлагается обновить КБЖУ, а не молча плодить дубли.
- Удаление продукта теперь с подтверждением.
- Экспорт backup стал надёжнее на iPhone: URL очищается с задержкой.
- После загрузки backup input сбрасывается, можно выбрать тот же файл повторно.
- Кнопкам добавлен `type="button"` там, где они не должны отправлять формы.
- Сохранена функция вставки продукта через JSON и старый парсер обычного текста.

Формат для вставки:
{"name":"Яйцо варёное","calories":155,"protein":12.6,"fat":10.6,"carbs":1.1}

Залей `index.html` вместо старого файла.
